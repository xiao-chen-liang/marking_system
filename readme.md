# 运行后端服务

```shell
python -m venv .venv

.\.venv\Scripts\activate

pip install -r requirements.txt

python backend\app.py
```

# 后端依赖

```shell
pip install pdfplumber

pip install openpyxl

pip install Flask

pip install flask-cors

pip install pandas

pip install mysql-connector-python
```

# 运行前端服务

```
npm install

npm run serve
```
